# 全网最穷站

> 一个属于穷人的互助社区。

## 这是干什么的？

分享失败经历、交流搞钱方法、抱团取暖的社区。
不卖课、不割韭菜、不贩卖焦虑——就是一群在贫困线上挣扎的人互相帮衬。

## 为什么要做这个站？

我叫何俊涛，血癌晚期，家里欠了300万。
我不想卖惨，但这就是现实。

在医院躺着的时候想：这世上应该有一个地方，让像我们这样的人能说说话。
不是鸡汤，不是营销号。就是互相看看——原来和我一样惨的人还有这么多，原来还有人比我更难也在撑着。

网站名字就叫"全网最穷站"——穷得坦坦荡荡。

## 技术栈

React 19 + TypeScript + Vite + Express + tRPC + MySQL + Tailwind CSS

但前提是——先帮到足够多的人。帮到人了，钱自然会来。
